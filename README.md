# Instagram Story Creative Generator

企画文からInstagramストーリーズ4枚構成のクリエイティブを作るためのWebアプリです。

ローカルで試せるほか、Vercel、Netlify、Firebase Hostingなどの静的ホスティングに公開すれば、メンバー全員が同じURLから利用できます。PWA対応済みなので、ブラウザからデスクトップアプリのようにインストールできます。

## できること

- `【1ページ目】` から `【4ページ目】` までの企画文を貼り付け
- CSV/XLSXから8〜10投稿分を一括読み込み
- 完成例に近い4枚構成へ自動レイアウト
- ピンク、イエロー、ブルーのテーマ切り替え
- 投稿ごとのテーマカラー自動割り当て
- 歯イラスト素材の登録
- 1〜3ページ目のイラストランダム割り当て
- Canvaで編集しやすいSVGを書き出し
- Canvaに読み込ませやすいPPTXを書き出し
- 投稿用確認に使えるPNGを書き出し
- Canva一括作成用CSVをコピー
- クライアント別テンプレート切り替え
- ブラウザへの自動保存
- Firebaseを使ったチーム共有保存
- PWAとしてインストール

## 使い方

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開きます。

## メンバー全員で使う方法

本番用に公開するときは、以下の流れがおすすめです。

```bash
npm install
npm run build
npm run preview
```

問題なければ `dist/` を静的ホスティングへ公開します。

おすすめの公開先:

- Vercel: GitHub連携で更新が簡単
- Netlify: ドラッグ&ドロップ公開も可能
- Firebase Hosting: Google系運用に寄せたい場合に便利

公開後は、メンバーにURLを共有してください。ChromeやSafariで開いたあと、ブラウザメニューから「アプリをインストール」または「ホーム画面に追加」を選ぶと、デスクトップアプリのように使えます。

Firebase設定を入れると、企画文、登録した歯イラスト、作成途中の状態をチーム共有の保存先に同期できます。設定がない場合は、各メンバーのブラウザ内保存だけで動きます。

クラウド保存の設定手順は [docs/firebase-setup.md](docs/firebase-setup.md) を見てください。

GitHubに接続しておくと、アプリ本体やクライアント別テンプレートの変更履歴を残せます。設定手順は [docs/github-backup.md](docs/github-backup.md) を見てください。

## Canvaでの編集

SVGを書き出してCanvaにアップロードすると、背景、図形、テキストを編集しやすい素材として扱えます。
4ページ目の写真部分はプレースホルダーなので、Canva上で医院写真に差し替えてください。

Canva上でテキスト編集したい場合は「PPTXを書き出し」を使い、CanvaでPowerPointファイルを読み込んでください。テキストはPowerPointのテキストボックスとして出力されるため、Canva側で編集できる可能性が高くなります。

## スプレッドシート一括読み込み

CSV、TSV、XLSX、XLSをアップロードできます。推奨の列名は以下です。

```text
投稿名,1ページ目,2ページ目,3ページ目,4ページ目
```

既存の運用シートのように、以下の列でも読み込めます。

```text
番号,内容,参考/ワード
```

この場合は `内容` を投稿名、`参考/ワード` を企画文として読み込みます。`参考/ワード` には `【1ページ目】` から `【4ページ目】` までの文章をまとめて入れてください。

1行を1投稿として読み込みます。読み込み後、投稿を選択して個別確認でき、全投稿分を1つのPPTXとして書き出せます。

テーマカラーは読み込み時に `ピンク → イエロー → ブルー` の順で自動割り当てされます。投稿を選択した状態でテーマボタンを押すと、その投稿だけテーマを変更できます。シートに `テーマカラー`、`カラー`、`色` の列がある場合は、そこに `ピンク`、`イエロー`、`ブルー` を入れて個別指定できます。

## 歯イラスト素材

左側の「歯イラスト素材」からPNG、JPG、SVGを複数登録できます。登録した素材はブラウザのローカルストレージに保存され、1〜3ページ目へ手動選択またはランダム割り当てできます。4ページ目は写真と予約導線のCTAページなので、歯イラストは入れません。

SVG素材は登録時に白背景をできるだけ除去し、余白を自動トリミングしたうえで、透明な正方形キャンバスに収めて透過PNGとして保存します。すでに登録済みの素材は、画面上の「登録済み素材を透過・サイズ調整」ボタンで再処理できます。

8〜10投稿分を作る場合は、投稿ごとに「1〜3ページをランダム割り当て」を押してから書き出すと、同じイラストの連続使用を減らせます。

全投稿PPTXを書き出す場合は、登録済みイラストから投稿ごと・ページごとに自動で別イラストを割り当てます。素材数が足りない場合は循環して使用します。

## クライアントを増やす方法

新規クライアントを追加するときは、以下を追加します。

- 完成デザイン例
- 企画文サンプル
- 背景画像、写真、固定素材
- テーマカラー
- 4ページ分のSVGレイアウト
- Canva編集用PPTXレイアウト
- 一括読み込み時の確認

詳しい手順は [docs/client-template-guide.md](docs/client-template-guide.md) を見てください。
